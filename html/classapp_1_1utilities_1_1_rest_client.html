static JSONArray connection ( String  url,
JSONObject  json,

)
inlinestatic

Connects to a specific URL, sending a JSON string and retrieving a JSON string in return.

Parameters
urlURL address containing the PHP script.
jsonJSON object to send to the PHP specified in the URL. If no input is required for database connection, can be null.
activityActivity object to display toast in case of failure.
Returns
JSON array returned from PHP call to the database. If nothing is returned from the PHP query, return null. On failed connection to the database, returns null and displays a toast to the app with the exception.
